# 3-Column Preview Card Component

This is a responsive web development challenge to build out a 3-column preview card component using HTML and CSS.

## Challenge Description

The challenge is to get the 3-column preview card component looking as close to the provided design as possible. Users should be able to view the optimal layout depending on their device's screen size and see hover states for interactive elements.

## Tools and Technologies

For this challenge, CSS Flexbox is used to create a responsive layout that adapts to different screen sizes. Media queries are utilized to adjust the design elements based on the viewport width, ensuring a consistent user experience across devices.

## Challenge Details

- CSS Flexbox model used for layout
- Media queries for responsive design
- Challenge level: Beginner
